Meridiano – TP1 Frontend II
Propósito

Este proyecto corresponde al Trabajo Práctico 1 de Frontend II (UADE).
Se trata de la implementación inicial del sitio web de Meridiano Café de Especialidad, basado en el brief y las user stories definidos previamente.

El objetivo es diseñar un template web semántico, responsive y accesible, que permita mostrar la propuesta de la marca, sus servicios y facilitar el contacto con clientes.

¿Cómo ver el proyecto?

Descargar el repositorio/carpeta completa.

Abrir el archivo index.html en cualquier navegador.

Navegar con la barra superior a las secciones Inicio, Proyecto y Contacto.



Funcionalidades incluidas

Navegación entre páginas (index.html, proyecto.html, contacto.html).

Layout responsive con CSS Grid y Flexbox.

Accesibilidad mínima:

Textos alternativos en imágenes (alt).

Formularios con label/for y campos requeridos.

Contraste legible y foco visible.

Formulario de contacto conectado a Formspree (Extra A).
